# diostatus — The disclose.io Maturity Model

diostatus defines a 6-level progression (Level 0 through Level 5) that describes how prepared an organisation is to receive and handle external vulnerability reports — from "no findable contact" at Level 0 to "full safe harbor with a public coordinated-disclosure timeline" at Level 5.

The model is deliberately plain-English. It's designed to be legible to non-security executives, useful as a self-assessment checklist, and specific enough that an external observer can look at an organisation's published posture and place it on the scale.
